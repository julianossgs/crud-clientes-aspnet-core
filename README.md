# CRUD Clientes - ASP.NET Core MVC

Projeto de estudo para prática de desenvolvimento web com ASP.NET Core MVC e MySQL.

## Tecnologias utilizadas
- C#
- ASP.NET Core MVC
- Entity Framework Core
- MySQL

## Funcionalidades
- Cadastro de clientes
- Edição
- Exclusão
- Listagem

## Objetivo
Projeto desenvolvido como preparação para testes técnicos de vaga Full Stack .NET.
